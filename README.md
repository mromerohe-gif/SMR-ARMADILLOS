# SMR-ARMADILLOS

## Objetivo del ejercicio

El objetivo de esta práctica es aprender a trabajar en grupo utilizando GitHub. Con este ejercicio hemos aprendido cómo compartir un repositorio entre varias personas y cómo organizar los cambios que hace cada miembro del equipo sin que haya problemas.
Cada integrante del grupo ha creado una carpeta con su nombre dentro del repositorio y ha añadido un archivo donde explica brevemente por qué le gusta la informática o qué es lo que más le llama la atención de este campo.

## Comandos utilizados

Para realizar esta actividad hemos utilizado varios comandos básicos de Git:
- git clone para descargar el repositorio en nuestro equipo.
- git pull para actualizar el repositorio con los cambios de otros compañeros.
- git add para añadir archivos al repositorio.
- git commit para guardar los cambios realizados.
- git push para subir los cambios al repositorio de GitHub.

## Conclusión 

Con esta actividad hemos aprendido cómo trabajar con un repositorio compartido y cómo gestionar cambios cuando varias personas trabajan en el mismo proyecto. También hemos visto la importancia de la organización y la comunicación cuando se trabaja en equipo utilizando herramientas como GitHub.
